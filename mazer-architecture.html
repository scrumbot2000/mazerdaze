Ingest daily notes + session transcripts. Deduplicate. Stage candidates in short-term recall store. Record light-phase signals.

😴 REM SLEEP — Phase 2
Read all staged entries from past 7 days. Extract recurring themes by concept frequency. Record REM reinforcement signals. Generate narrative diary entry.

🌙 DEEP SLEEP — Phase 3
Score every candidate using 6 weighted signals: Relevance (0.30) · Frequency (0.24) · Query diversity (0.15) · Recency (0.15) · Consolidation (0.10) · Richness (0.06). Must pass minScore + minRecallCount + minUniqueQueries gates.
